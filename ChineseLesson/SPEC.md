# Chinese Lesson App

Single-page React 18 learning game with:
- Russian-language UI for children.
- 5 lessons (food/family/animals/numbers/colors).
- Vocab screen, fill-in-the-blank exercises, scoring, result screen.
- Lesson editor with words/exercises/JSON tabs.
- Web Audio API SFX + Google Translate TTS fallback to Web Speech API.
- Inline style objects + in-app keyframe animations.

Run by opening `index.html` in a browser.
