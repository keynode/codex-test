# Chinese Lesson App

Standalone React 18 learning app for kids under 9.

Files:
- `index.html` — standalone production page with React UMD + Babel
- `lesson-data.json` — example lesson data format
- `app.js` / `styles.css` — old prototype files, not used by the standalone build

Main implemented features:
- lesson menu
- vocabulary screen
- fill-in-the-blank exercise flow
- result screen
- lesson editor
- Web Audio API sounds
- Google Translate TTS with Web Speech fallback
- confetti and floating emoji effects
